一、vue框架

一、安装(node.js )

浏览器搜索，直接去官网下载安装即可，安装时无脑下一步

node官网地址

二、安装npm(存在网络原因可以安装cnpm)

npm全称为 Node Package Manager,是一个基于№ode.js的包管理器,也是整个№ode.js社区最流行、支持的第三方模块最多的包管理器

npm install -g cnpm –registry=https://registry.npmtaobaoorg

三、安装vue-c1i(脚手架)

或者cnpm install -g @vue/cli

npm i -g @vue/cli-init

四、安装 webpack

cnpm install -g webpack
webpack是 JavaScript打包器( module bundler)

五、创建项目

1、图形化界面创建项目
  • cmd中输入vue ui (电脑自动跳转到浏览器的本地图形化地址，在此创建项目)
2、命令行创建项目

  • 在命令行中进入项目的根目录

  • 输入命令：vue init webpack 项目名

    或者：vue create 项目名

六、运行项目

npm run dev
