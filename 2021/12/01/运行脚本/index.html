一、手机脚本运行

一、基础环境

  • 下载adb调试桥工具包下载地址
  • 配置环境变量：(下载的工具包的位置)我的是：D:\platform-tools
  • 手机打开开发者选项，允许调试，并连接电脑
  • 在CMD中运行：adb devices
    • 结果出现unauth…即手机没有允许调试
    • 结果出现devices 即手机已经允许调试

二、模块介绍

xldr模块

xldr为python第三方模块，用来读取Excel表格数据

安装命令：pip install xlrd

xldr常用属性方法
1、操作行
2、操作列
3、操作单元格

xlwt模块

xlwt模块为python第三方模块，用于写入Excel表格数据

cmd安装命令： pip install xlwt

xlwt常用属性方法
1、基础用法
2、写入格式化

xlsxwriter模块

xlsxwriter为python第三方模块，用于向生成的Excel表格插入数据、图表等操作

cmd安装命令：pip install xlsxwriter

xlsxwriter常用属性方法
1、基础用法
2、写入格式化
3、写入图表到Excel

python-docx模块

python-docx为Python第三方模块，用于自动化生成和修改Word文档

cmd中安装：pip install python-docx

python-docx常用属性方法
1、基础用法
2、写入表格和图片
3、Word样式处理
4、Word转PDF

先安装：pip install pywin32

python-pptx模块

python-pptx为python第三方模块，用于自动化生成和更新PowerPoint(.pptx)文件

CMD中安装：pip install –index-url https://mirrors.ustc.edu.cn/pypi/web/simple/ python-pptx

python-pptx常用方法属性
1、基础用法
2、添加图形到PPT
3、添加表格到PPT
